# Arquitetura do Sistema

O PTrab Inteligente é construído com tecnologias modernas para garantir performance e segurança.

## Frontend
- **React + TypeScript:** Interface robusta e tipada.
- **Tailwind CSS:** Estilização responsiva e padronizada.
- **Shadcn/UI:** Componentes de interface de alta qualidade.

## Backend e Dados
- **Supabase:** Banco de dados PostgreSQL com autenticação integrada.
- **Edge Functions:** Processamento server-side para consultas externas (Combustíveis, PNCP).
- **Row Level Security (RLS):** Garantia de que cada usuário acesse apenas seus próprios dados.
