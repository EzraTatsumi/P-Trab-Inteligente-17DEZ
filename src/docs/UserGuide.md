# Guia do Usuário - PTrab Inteligente

Bem-vindo ao PTrab Inteligente, a ferramenta definitiva para planejamento e gestão de custos operacionais e logísticos.

## 1. Gerenciamento de PTrabs
No painel principal, você pode criar novos Planos de Trabalho (PTrabs), clonar variações existentes ou consolidar múltiplos planos em um só.
- **Minutas:** Todo PTrab nasce como uma "Minuta". Após a revisão, ele pode ser aprovado e receber uma numeração oficial.
- **Compartilhamento:** Você pode gerar links de compartilhamento para que outros usuários solicitem acesso de colaboração ao seu PTrab.

## 2. Resumo de Custos
Dentro de cada PTrab, o painel lateral exibe um resumo em tempo real dos custos:
- **GND 3 (Custeio):** Soma de logística e operacional.
- **GND 4 (Investimento):** Materiais permanentes.
- **Créditos Disponíveis:** Você pode inserir o teto de gastos para monitorar o saldo restante.

## 3. Itens Operacionais e Logísticos
### Horas de Voo (AvEx)
Módulo específico para o planejamento de horas de voo da Aviação do Exército, permitindo o cálculo preciso baseado no tipo de aeronave e amparo legal.

### Serviços de Terceiros e Locações
Gestão de contratos de serviços e locação de estruturas, com suporte a detalhamento customizado para cada item.

### Integração API PNCP
Ao cadastrar itens de Material de Consumo ou Permanente, você pode utilizar o sistema de busca via API do PNCP (Portal Nacional de Contratações Públicas) para importar dados de Atas de Registro de Preços (ARPs) vigentes, garantindo valores de mercado atualizados.

## 4. Editor DOR (Documento de Operacionalização de Recursos)
O Editor DOR permite gerar o documento formal de solicitação de recursos.
- **Grupos de Itens:** Organize os custos do PTrab em grupos específicos para o documento.
- **Motivação e Consequência:** Campos dedicados para a justificativa técnica da operação.

## 5. Exportação e Impressão
Gere relatórios em PDF ou planilhas Excel detalhadas com toda a memória de cálculo do seu planejamento.
