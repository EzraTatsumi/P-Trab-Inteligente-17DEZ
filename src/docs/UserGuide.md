# Guia do Usuário

Aprenda a utilizar as principais ferramentas do PTrab Inteligente.

## Gerenciamento de P Trabs
- **Criação:** Inicie como "Minuta" para rascunho.
- **Aprovação:** Atribua um número oficial para finalizar a edição básica.
- **Clonagem:** Use para criar variações de um mesmo planejamento sem perder os dados originais.

## Preenchimento de Necessidades
- **Navegação:** Escolha entre as abas Logística e Operacional.
- **Catálogo de Diretrizes:** Utilize a busca por CATMAT ou UASG para encontrar itens homologados.
- **Memória de Cálculo:** O sistema gera justificativas automáticas, mas permite edição manual para casos específicos.

## Configurações de Custos
- **Anos de Referência:** Gerencie diretrizes para diferentes anos fiscais.
- **Importação PNCP:** Importe atas de registro de preços diretamente do portal nacional.
- **Importação Excel:** Cadastre subitens e itens em massa utilizando planilhas padrão.

## Relatórios e Exportação
- **Impressão:** Gere PDFs nos formatos oficiais do Exército.
- **Excel:** Exporte dados brutos para análises externas.
- **DOR:** Gere o Documento de Operacionalização de Recursos consolidado.

## Colaboração
- **Compartilhamento:** Convide outros usuários para editar o mesmo P Trab via link.
- **Vinculação:** Solicite acesso a documentos de terceiros através do menu de configurações.
