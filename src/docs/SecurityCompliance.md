# Segurança e Conformidade

Seguimos as melhores práticas de proteção de dados.

## Autenticação
- Acesso restrito via e-mail e senha com criptografia de ponta a ponta.
- Sessões gerenciadas de forma segura pelo Supabase Auth.

## Integridade dos Dados
- Backups automáticos realizados pelo provedor de infraestrutura.
- Auditoria de alterações em registros críticos.

## Privacidade
- Isolamento completo de dados entre diferentes usuários e organizações, exceto quando explicitamente compartilhado.
